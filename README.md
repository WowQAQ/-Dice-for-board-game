# Dice-for-board-game
由于玩桌游时用网页或者小程序掷出的骰子常常离谱让人直呼有内幕    
于是自己写了一个  
可以查看历史掷出的次数以防有人输了说概率有问题^^  
----------------------------------------------------------  
使用方法  
打开.exe回车掷骰子
----------------------------------------------------------
v2.0更新
又有人说概率有问题了，于是游戏开始前先掷一万次骰子并显示概率分布
其他更新内容包括：
1.每一局为固定一个随机种子
2.开始前可以选择一次丢几个骰子，以适应各种桌游
